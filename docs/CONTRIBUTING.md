# Contribuer à CoupeMAP

Merci de votre intérêt pour CoupeMAP ! Les contributions sont les bienvenues.

## 🔀 Dépôt principal

Le dépôt **upstream** (référence) est :
**[github.com/ozantokman/coupemap](https://github.com/ozantokman/coupemap)**

Toutes les Pull Requests doivent être soumises vers ce dépôt.

## 📋 Comment contribuer

1. **Forkez** le dépôt sur GitHub
2. **Clonez** votre fork : `git clone https://github.com/VOTRE_NOM/coupemap.git`
3. **Créez une branche** : `git checkout -b feature/ma-fonctionnalite`
4. **Faites vos modifications** et testez localement
5. **Committez** : `git commit -m "feat: description courte"`
6. **Poussez** : `git push origin feature/ma-fonctionnalite`
7. **Ouvrez une Pull Request** vers `ozantokman/coupemap:main`

## ✅ Règles

- **Crédits** : Ne supprimez pas les crédits de l'auteur original (Ozan Tokman) du code, de l'interface utilisateur ou de la documentation
- **Statique** : L'application doit rester déployable en tant que site statique (pas de serveur backend obligatoire)
- **Encodage** : Tous les fichiers doivent être en UTF-8 (sans BOM)
- **Langue** : L'interface est en français. Les commentaires de code et la documentation peuvent être en français ou en anglais
- **Tests** : Testez vos modifications sur Chrome, Firefox et Safari avant de soumettre

## 🐛 Signaler un bug

Ouvrez une [Issue](https://github.com/ozantokman/coupemap/issues) avec :
- Description du problème
- Étapes pour reproduire
- Navigateur et système d'exploitation
- Capture d'écran si possible

## 💡 Proposer une fonctionnalité

Ouvrez une [Issue](https://github.com/ozantokman/coupemap/issues) avec le tag `enhancement` et décrivez :
- Le besoin utilisateur
- La solution proposée
- Les alternatives envisagées
